# Cyber Fingerprint 🐾 

Calculate fingerprints of a website for OSINT search.

## Installation

```bash
git clone https://github.com/jmousqueton/cyberfingerprint.git

pip install -r requirements.txt

cd frontend
npm install
npm run build
```

## Dev server

```bash
uvicorn app:app --reload
```
